## 向量 (Vector)
顾名思义，即“有方向的量”，用中学教材的说法就是--同时具有大小与方向的量，与其相对的就是**标量(scalar)** 。

向量本身没有什么特殊的性质，也容易理解，需要留意的是，如同数轴有单位刻度一样，向量中的每个**元素(element)** 可以理解为n个**基向量(basis)** 的叠加。
